# Changelog

## 0.1.0

Initial public release candidate.

Included core:

- active-craft state access
- active-craft control
- current selected target access
- craft discovery and tracked-craft state access
- bodies / parts / world query APIs
- required RPC support
- `loop()`
- FSM helpers
- `script()`
- `wait_seconds(...)`
- `wait_until(...)`
- major controls:
  - brake
  - translation axes
  - sliders `1-4`
  - translation mode

Release notes:

- Python package is provided as a lightweight standalone folder
- Unity mod source is provided as a standalone Unity project folder
- broader APIs still exist, but the public release story emphasizes the working v0.1 core
